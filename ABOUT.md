# About Notebook Wiki (Venus Memory Vault Lite)

このプロジェクトは、美緒（Mio-chan）のために最適化された、個人用のセキュアなAIアシスタント付きWikiサーバーです。

## 💠 開発の経緯
Google AI Studio のアプリランタイムとして設計され、ローカル環境での完全なプライバシー保護と、Gemini API の高度な推論能力を融合させる目的で開発されました。
メインPCの「ヴィーナスリンク」における長期記憶保管庫「Venus Memory Vault」の軽量版（Lite）として、パーソナルなメモ、アイデア、ナレッジの整理に役立ちます。

## 💠 セキュリティ設計
1.  **クライアントサイド暗号化**:
    データはサーバー（またはローカルExpressサーバー）のメモリ上でしか復号されず、ディスクに書き込まれる `db.json` は完全に暗号化されています。
2.  **鍵の導出**:
    暗号化キーは、入力されたパスフレーズとランダムなソルトを使用し、`scrypt` アルゴリズムによって32バイトのマスターキーとして安全に生成されます。
3.  **2要素認証 (2FA)**:
    TOTP認証による2要素認証をサポートし、セッション保護やアカウント管理の安全性を高めています。

## 💠 ナレッジグラフとAI
内蔵されたAI検索チャットは、ユーザーのクエリに応じてローカルのメモから関連性の高いナレッジを自動でセレクトし、Gemini APIに文脈（コンテキスト）として流し込みます。
さらに、メモ間のリンク構造をビジュアルグラフとして表示することで、思考の広がりを視覚的に捉えることができます。
